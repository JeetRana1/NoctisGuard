# NoctisGuard

Web frontend for NoctisGuard.

## Quick start
- Open `index.html` in the browser to preview the static site.
- This project contains an optional Node server (`server.js`) and `package.json`.

## Deployment
- Recommended: Deploy the static site on Vercel by connecting this GitHub repository (Vercel will serve `index.html`).
- If you need a persistent Node server, consider using Render, Fly, or Railway, or convert endpoints in `server.js` to Vercel serverless functions.

## Files of interest
- `index.html` — main site
- `server.js` — optional Node server
- `package.json` — project metadata
